Mandragora Code Studio

El grupo de desarrollo indi Mandragora Code Studio nace con la intención de crear videojuegos de alta calidad para Linux, Mac y Windows. Todos nuestros juegos serán Opensource y usaremos tecnologías libres.

Nuestra denominación como grupo Indi indica que no contamos con financiación externa pero si de verdadera pasión por el arte de la creación de videojuegos. Esta pasión nos lleva a querer hacer algo grande, algo que como individuos no podríamos lograr, pero como grupo sabemos que lo podemos hacer. Para eso nace Mandragora Code Studio.

Gonzalo R. Díaz & Jose V. Infantes, fundadores.
